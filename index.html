Contenedores: div vs span

<div> es un contenedor de bloque que se utiliza para agrupar elementos y aplicar estilos o scripts a un bloque completo, es decir, ocupa todo el ancho disponible. Por ejemplo:

<div class="caja">

<span> es un contenedor en línea que se utiliza para agrupar elementos de texto y aplicar estilos o scripts a un fragmento específico, es decir, no ocupa todo el ancho disponible solamente lo que ocupe el elemento . Por ejemplo:

<span class="resaltado">Texto resaltado</span>
